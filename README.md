# Online Gaming Store Database

## Project Overview
This project was created as part of a university database project. It focuses on designing and implementing a MySQL relational database system for an online gaming store. The system efficiently manages players, games, developers, purchases, and reviews, allowing the store to seamlessly track game sales, user activity, and game ratings.

## Tools Used
- MySQL (via MySQL Workbench)

## Features
- Player and account management
- Game cataloging
- Developer directory
- Transaction and order tracking
- Feedback and rating system
